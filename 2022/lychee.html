Planting Lychee Gallary on Padavan Firmware

Mar 4 2022 router #shell

#Hardware

  • Old router NEWIFI-MINI. I had this only router with a USB port.
  • EXT-4 formatted USB drive.

#Settings

  1. USB Programs->Enable Optware->Entware
  2. Extensions->force opt install->Install to USB drive
    • Run df -h. If /opt was not mounted to USB drive, run mount \dev\sda1 \opt to mount manually. Here \dev\sda1 is my USB drive. It might be different, just change the directory accordingly.
      It's gonna take a while to install the opt. A cuppa is highly recommended.
  3. Make a swap

    • The memory of NEWIFI-MINI is only 64MB, SQL service will not run properly on it.

    • Swap will not mount itself after rebooting. Add swapon /opt/swap to the scripts to run after booting to mount the swap file automatically.

  4. Web Environment->Enable nginx+mysql+php

    • Here I ran into 2 problems: No nginx, No mysqld. The solution is simple as well. run the commends below.

      Everything will be fine now.

    It takes a while, too. Wait until the information appearing in the log area.

  5. Can't download phpmyadmin. Take a glimpse at the log area to comfirm the version, and download the zip manually on laptop, then transport it into the router with SSH.
  6. mysql rebooting continuously. Try running mysql in ssh terminal. It says there might be some database missing. Change directory to /opt/bin and run mysql_install_db. This will re-initialize the mysql database, and everything will be fine.
  7. phpmyadmin might fail. Try replacing $cfg['Servers'][$i]['host'] = 'localhost'; with $cfg['Servers'][$i]['host'] = '127.0.0.1';
